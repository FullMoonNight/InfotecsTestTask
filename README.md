# InfotecsTestTask

Demo:   https://fullmoonnight.github.io/InfotecsTestTask/

Тестовое задание на языке *JS* на вакансию стажера в компанию Infotecs

### Инструкция
1. Для начала установить все зависимости через терминал
> **npm install**
2. Собрать проект при помощи webpack используя одну из следующих команд
> **npm run prod**
> > Данная команда создаст папку dist, в которой будут лежать все собранные и минифицированные файлы. Запускать html

> **npm run dev**
> > То же, что и предыдущая, но файлы не минифицированные. Запускать html

> **npm run server**
> > Соберет проект и запустит проект в браузере
3. Так же для просмотра demo работы выше имеется ссылка
